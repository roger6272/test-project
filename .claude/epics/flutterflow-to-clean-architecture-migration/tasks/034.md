```markdown
# Privacy Policy for Trackwise

Last updated: [Date]

## Introduction
Trackwise ("we", "our", "us") respects your privacy and is committed to protecting your personal data.

## Data We Collect
- Email address (for authentication)
- Item names and counts (user-created data)
- Event logs (timestamp and count data)
- Usage analytics (anonymized)

## How We Use Your Data
- To provide app functionality
- To sync data across devices
- To improve app performance
- To send data export emails (when requested)

## Data Storage
- All data is stored in Google Firebase Firestore
- Data is encrypted in transit and at rest
- Data is retained until account deletion

## Your Rights (GDPR)
- Right to access your data (via in-app export)
- Right to delete your data (via in-app account deletion)
- Right to data portability (via JSON export)

## Third-Party Services
- Firebase (Google) - data storage and authentication
- Google Analytics - anonymized usage statistics

## Contact
Email: [your-email@example.com]

## Changes to Policy
We may update this policy. Changes will be posted in the app.
```

**Privacy Policy Viewer**:
```dart
class PrivacyPolicyPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Privacy Policy')),
      body: FutureBuilder<String>(
        future: rootBundle.loadString('assets/privacy_policy.md'),
        builder: (context, snapshot) {
          if (snapshot.hasData) {
            return MarkdownWidget(
              data: snapshot.data!,
              padding: const EdgeInsets.all(16),
            );
          }
          return const Center(child: CircularProgressIndicator());
        },
      ),
    );
  }
}
```

```
Trackwise - Habit Tracker with ESP32 Integration

Track your daily habits with physical buttons! Trackwise connects to ESP32 hardware, letting you log events with a simple button press.

FEATURES:
```
